🚀 5-Minute Quickstart

Bare minimum to get going. See each section below for details.

1

Drag onto the "📂" zone on the home screen. Multiple files OK.
2
WASD to move · Right-drag to look around

First-time checklist

  • Press M to enter measure mode
  • Manage the scene from the Layer panel (left)
  • All major features live in the bottom centre cbar
  • Lost? Hit the ❓ button to open this manual

📂 Open a file

The drop zone on the home screen (right after launch) is your entry point.

📁 Supported formats

  • 3D Gaussian Splatting: .ply .splat .spz .ksplat .rad .sog
  • 3D models: .obj .glb .gltf .fbx
  • Project: .zip (saved from LOCAHUN 3D)

📥 How to load

  • Click → file picker (Ctrl-click for multi-select)
  • Drag & drop (multiple files at once OK)
  • First file becomes the main scene; the rest are added as layers
💡 Tip: Large PLYs (100MB+) take tens of seconds to load. A progress bar appears while loading.

🎮 Move the camera

Keyboard for forward/back/left/right/up/down · mouse to look around.

Speed

  • Hold Shift for 5× speed
  • Mouse wheel changes the cruise speed itself
  • Current speed is shown in the HUD (top-left)

🔄 Fix scene orientation

PLYs from different exporters may appear upside-down or rotated.

🎯 Upside-down / sideways

  1. Select the layer
  2. In the right panel, try the Flip X / Y / Z Axis buttons
  3. For finer control use the rotation number fields below

🔁 Fine adjustment

  • Try the X / Y / Z flip toggles (3 axes, independent)
  • If still off, dial in Position / Rotation / Size manually

📁 Layer panel

The "Scene Layers" sidebar on the left is the hub. Every loaded file or object lives here.

  • 👁 show / hide
  • 🗑 delete (Z to undo, up to 10 steps)
  • Double-click rename
  • Drag reorder / nest into folders
  • Ctrl + click multi-select → Delete to remove all

⇆ Position / Rotation / Scale

Selecting a layer reveals the "Transform" panel at the bottom of the left sidebar.

  • Numeric X / Y / Z input (position / rotation / scale)
  • Drag the arrow handles in the 3D view directly
  • Flat handle (□) for XZ-plane translation
  • Rotation rings (↻ X/Y/Z) for axis rotation
  • Cube handles for per-axis scaling
💡: Z undoes up to 10 steps. Y redoes.

＋ Add objects

From the top-left "Add Object ▾" menu:

  • ＋ 3DGS — additional splat file (PLY/SPLAT/SPZ/KSPLAT)
  • ＋ OBJ/GLB — 3D model (OBJ/GLTF/GLB/FBX)
  • 📦 Cube / ⚽ Sphere / 📁 Folder / 💡 Light / 🟠 Event — scene helpers

📐 Distance measure

Measure the real distance between two points on a splat surface.

  1. Press M or click "📐 Measure" in the cbar to enter measure mode
  2. Left-click hold → a ghost marker tracks the cursor
  3. Release to commit point A
  4. Repeat to set point B → distance shown on screen
  5. Drag the X/Y/Z arrows on each marker to fine-tune
📏 Accuracy: Snaps to the nearest splat surface point automatically. Up to 2 million points are cached for high-precision detection in large scenes.

📷 Camera tool (previz)

Set lens / sensor like a real camera and export JPEGs. Blender Photographer-style.

Panel layout

Right panel — technical settings

  • 🔍 Lens focal length (14 / 35 / 50 / 85 / … presets)
  • ⬛ Sensor FF / APS-C / MFT / S35 / 65mm / Custom
  • 🎯 Angle pan / tilt / roll (Dutch angle)
  • ☀ WB colour temperature K (2000–12000)
  • 🎬 Aspect 16:9 / 2.39 / 1:1 / Custom
  • # Grid rule-of-thirds / golden / safe / custom (multi-select OK)

Bottom-left — shot info

  • 📷 Camera info live readout
  • 📝 Meta shot name / rig / env / notes
  • 📸 Capture burn-in toggle + quality + shutter
  • 📥 Salvage drop a past JPEG to restore camera state

Sensor & lens relation

FOV is computed automatically from focal length + sensor (the same pinhole formula real cameras use):

  • FF 36×24mm + 35mm lens → 54.4° H / 38.6° V
  • To match this on APS-C: 35÷1.5 ≈ 24mm
  • The HUD always shows the 35mm-equivalent as well

Safe frame & margin

The chosen aspect is drawn as a frame; the outside is masked dark. The "Outer margin" slider insets the frame.

📥 Salvage (restore camera state)

JPEGs captured by LOCAHUN 3D embed full camera state into a JPEG COM segment.

  1. Open the camera tool
  2. Drop a past JPEG onto the dashed area at the bottom of the bottom-left panel
  3. Camera position / orientation / lens / WB / grid settings all restore
⚠ Note: Re-saving the JPEG in another app may strip the embedded data. Keep the original capture file.

🌅 Environment (pseudo HDRI)

Toggle from "🌅 Env" in the cbar. Changes the sky colour and global tint at once.

  • 9 presets: ☀ Day / 🌅 Morning / 🌇 Sunset / 🌃 Night / ☁ Overcast / ⛅ Hazy / 🌧 Rain / 🌌 Twilight / Off
  • Rotation slider rotates the sky 0–360° (align light direction)
  • Intensity slider 0–200% (100% standard, 200% emphasises lighting)

Splats and models receive a multiplicative tint — night turns deep blue, sunset goes warm.

🗺 Minimap

Toggle the minimap from the cbar's "🗺 Map" button.

  • Right-click hold on the map → release to teleport
  • Range: 10 / 50 / 100 / 500 m (default 100m)
  • Click to expand the minimap view

🖥 Quality & speed

Toggle render quality from the cbar's "🖥 Quality" button.

  • Low = fast (for big PLYs / weak GPUs)
  • Medium / High = normal use
  • While the camera moves, resolution drops to 65 % automatically and restores on idle (Spark sort-lag mitigation)

💾 Save project

  • Top-left 📁 Save button → ZIP download (full scene + camera state)
  • Ctrl+S works too
  • Top-left 📥 Open or the home screen "📦 Open project" to restore
  • "▶ Continue editing" (home screen) keeps your previous state

❓ FAQ

Q. The loaded scene is upside down

A. Select the layer → try the X / Y / Z axis flip buttons under "View adjust".

Q. The screen edges go black / smear when I pan

A. Spark sort-lag. Resolution drops automatically while moving and restores on idle. If it still bothers you, drop the cbar's "🖥 Quality" one notch.

Q. The measure tool punches through the surface to the back

A. Cache density is already maxed (2M points). If still off, verify the active layer actually contains the main splat.

Q. Salvage doesn't restore my JPEG

A. The image was likely re-encoded by another app. Only the original capture from LOCAHUN 3D is supported.

Q. No aperture or ISO?

A. This is a virtual camera — no DOF or exposure simulation. Only WB (colour temperature) affects the look.

Q. How do I open the manual again?

A. The "❓" button at the bottom-centre of the screen opens it any time.
