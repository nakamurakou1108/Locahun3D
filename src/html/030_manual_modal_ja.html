🚀 5 分で使う

ロケハン3Dは、3Dスキャン（3DGS）を読み込んでロケハン・絵コンテ・プリビズに使うビューアーです。まずはこの3ステップ。

  1. 読み込む — 最初の画面の「📂」エリアに 3DGS ファイル（.splat / .ply など）をドラッグ。試すだけなら「デモシーン」ボタン。
  2. 動かす — マウス右ドラッグで視点回転、WASDで移動。
  3. 使う — 画面下のボタンで「カメラ / 日照 / 測定」、上のボタンで「オブジェクト追加」。左のレイヤーパネルで管理。

🎮 カメラを動かす

  • 視点を回す：マウス右ドラッグ
  • 移動：W前 / S後 / A左 / D右 / Q下 / E上
  • 移動速度：マウスホイール（または画質パネルのスライダー）
  • 視点リセット：上部「🏠 カメラリセット」

スマホ / iPad では、左下のジョイスティックで移動、画面ドラッグで視点回転します。

📁 レイヤーとフォルダー

左の「シーンレイヤー」に、読み込んだファイルや追加オブジェクトが並びます。

  • 👁 表示 / 非表示の切替
  • ✏️ 名前を変更（ダブルクリックでも可）
  • 🗑 削除（Ctrl+Z で元に戻せます）
  • ドラッグで並べ替え・フォルダーへ収納
  • 🔒 フォルダーをロック：中身の移動・編集・削除をまとめて無効化（誤操作防止）。もう一度押すと解除。

⇆ 位置・回転・拡縮

オブジェクトを選ぶと、3Dハンドル（ギズモ）と数値入力が出ます。

  • 矢印＝移動 / リング＝回転 / キューブ＝拡縮
  • 「ローカル / ワールド」で座標系を切替（新規・読み込みは既定でローカル）
  • 位置・回転・拡縮・寸法の数値入力は 0.01 刻みで微調整できます
  • iPad / スマホではギズモ（矢印・リング・ピボット）を指でドラッグして操作できます

＋ オブジェクトを追加

上部「📦 オブジェクト追加」から種類を選び、シーン内を左クリック長押しして配置します（押して位置を探り、離して確定。Escで中止）。

  • 📦 Cube — 寸法の目安・あたり
  • 🟠 Event — 注目ポイント。画像を貼れます
  • 👤 Figure — 人型。身長を変えてスケール感・アイレベルを確認。関節を選んで回転、またはIKハンドルを引いてポーズ付けできます
  • 🛣 パス — 4点で囲む区画＋中央ラベル（駐車区画など）。ラベルは複数行OK

📐 距離測定

画面下「📐 測定」をON。スキャン表面の2点間の実距離を測ります。

  • 左クリック長押しで点をプレビュー → 離して確定（点A → 点B）
  • 「+ 高さを図る」で3点目（高さ）も測定
  • マーカーは矢印（軸）・XZ面で微調整、🗑 でクリア
  • iPad / スマホでも、矢印・マーカーを指でドラッグして微調整できます
  • 距離・高さ・成分は 0.01 m（1cm）単位で表示します

📷 カメラツール / アニメーション

カメラツール（下「📷 カメラ」）：実機のレンズ・センサー・アスペクト比を設定し、構図枠とグリッドを表示。JPEG撮影でカメラ情報も焼き込めます。

カメラアニメーション（上「🎞 アニメーション」）：カメラ位置を複数登録してパスを作り、MP4で書き出し。動きは「イージング（緩急） / リニア（等速）」を選べます。

☀ 日照・天気

画面下「☀ 日照」：日付・時刻・都道府県から太陽の位置と光のムードを再現します。

  • 時刻スライダーで朝〜夜の光をスクラブ
  • 天気（晴れ / 曇り / 雨）で空の雰囲気を変更
  • 「🌐 実際の天気予報を取得」でオンラインの実天気に追従（任意）

🚶 アバター歩行・📱 AR

アバター歩行（上「🚶 アバター歩行」）：人の目線でシーンを歩いて確認できます。WASDでアバターを移動、視点は通常どおり。実際に立ったときの見え方・アイレベル・スケール感のチェックに。もう一度押すと解除。

AR モード（上「📱 AR」・スマホ / タブレットのみ）：背面カメラの映像に 3DGS を重ねて表示し、端末を傾けて見回せます。現地で実寸の重なりを確認するときに。

※ iOS は初回に「設定 > Safari > モーションと画面の向きのアクセス」を ON にし、ページを再読み込みすると AR が使えます。

💾 保存・読み込み

  • Zip保存（左上）：シーン全体を1つのZIPに保存。レイヤー・オブジェクト・カメラ・設定をまるごと復元できます。
  • 読み込み（レイヤーパネル上の「📁 読み込み」）：ZIPや 3DGS / モデルを追加。
  • 🎥 カメラ保存：今のカメラ位置をレイヤーとして保存。

❓ よくある質問・ショートカット

  • シーンが上下逆 / 横倒し → レイヤーを選び「軸反転」または角度入力で調整。
  • 動作が重い → 右上の画質チップで「低」に。描画は最大60fpsです。
  • 言語 → 右上「EN / 日本語」で切替。

ショートカット：Ctrl+Z 取り消し / Ctrl+Y やり直し / Ctrl+D 複製

📂 ファイルを開く

ホーム画面 (起動直後) のドロップゾーンが入口です。

📁 対応形式

  • 3D Gaussian Splatting: .ply .splat .spz .ksplat .rad .sog

📥 読み込み方

  • 1 つ目はメインシーン、残りは追加レイヤー
💡 ヒント: 大きい PLY (100MB+) は読み込みに数十秒かかります。読み込み中はプログレスが表示されます。

🎮 カメラを動かす

キーボードで前後左右上下、マウスで視点回転。

速度を変える

  • Shift 押している間だけ 5 倍速
  • マウスホイールで巡航速度自体を変更
  • 右下に現在の速度が表示されます (HUD)

  3. それでもズレるなら下の角度入力で直接調整

  • X / Y / Z 軸反転 を組み合わせて試す (3 軸独立トグル)

📁 レイヤーパネル

左サイドの「シーンレイヤー」が中心。読み込んだファイルやオブジェクトはすべてここに並びます。

  • 👁 表示 / 非表示
  • 🗑 削除 (Z で取り消し可、最大10ステップ)
  • ダブルクリック 名前を変更
  • ドラッグ 並べ替え / フォルダ階層化
  • Ctrl+クリック 複数選択 → Delete で一括削除

⇆ 位置・回転・拡縮

レイヤーを選択すると左パネル下部に「Transform」が表示されます。

  • X / Y / Z の数値入力 (位置/回転/スケール)
  • 3D ビューに表示される矢印ハンドルを直接ドラッグでも動かせます
  • 水平面ハンドル (□) で XZ 平面移動
  • 回転リング (↻ X/Y/Z) でその軸まわりの回転
  • キューブハンドルで軸別スケール
💡: Z で操作を最大10回まで戻せます。Y で進む。

＋ オブジェクトを追加

左上の「オブジェクト追加 ▾」メニュー から:

  • ＋ 3DGS — 追加の splat ファイル (PLY/SPLAT/SPZ/KSPLAT)
  • ＋ OBJ/GLB — 3Dモデル (OBJ/GLTF/GLB/FBX)
  • 📦 キューブ / ⚽ 球 / 📁 フォルダ / 💡 ライト / 🟠 イベント — シーン構成用のヘルパー

📐 距離測定

splat 表面上の 2 点間の実距離を測ります。

  1. M または cbar の「📐 測定」で測定モードに入る
  2. 左クリック長押し → カーソル先にゴーストマーカーが追従
  3. 離した瞬間に点 A が確定
  4. もう一度同じ操作で点 B → 距離が画面上に表示
  5. マーカーを矢印 (X/Y/Z) でドラッグして微調整可
📏 精度: splat の表面の最近接点を自動でスナップします。シーンサイズに応じて 最大 200 万点をキャッシュして高精度に判定。

📷 カメラツール (プリビズ機能)

実機カメラのようにレンズ・センサーを設定して JPEG 出力。Blender Photographer 風。

パネル構成

右パネル — 技術設定

  • 🔍 レンズ 焦点距離 (14 / 35 / 50 / 85 / … プリセット)
  • ⬛ センサー FF / APS-C / MFT / S35 / 65mm / カスタム
  • 🎯 角度 パン / ティルト / ロール (ダッチアングル)
  • ☀ WB 色温度 K (2000–12000)
  • 🎬 アスペクト 16:9 / 2.39 / 1:1 / カスタム
  • # グリッド 三分割 / 黄金 / セーフ / カスタム (複数 OK)

左下パネル — ショット情報

  • 📷 カメラ情報 リアルタイム表示
  • 📝 メタ ショット名 / 機材 / 環境 / 備考
  • 📸 キャプチャ バーンインON/OFF + 画質 + 撮影
  • 📥 サルベージ 過去JPEGをドロップで復元

センサーとレンズの関係

焦点距離 + センサーサイズから視野角 (FOV) が自動計算されます (実機と同じピンホール式):

  • FF 36×24mm + 35mm レンズ → 水平 54.4° / 垂直 38.6°
  • APS-C で同じ画角を得たい場合: 35÷1.5 ≈ 24mm
  • HUD には常に 35mm 換算も表示

セーフフレームと余白

選択したアスペクトの枠が表示され、外側が暗くマスクされます。「外側余白」スライダで枠を内側にオフセット。

📥 サルベージ (カメラ位置の復元)

ロケハン3Dで撮った JPEG にはカメラ位置・角度・全設定が JPEG COM セグメント に埋め込まれています。

  1. カメラツールを起動
  2. 左下パネル下部の点線エリアに以前撮った JPEG をドラッグ
  3. カメラ位置 / 向き / レンズ / WB / グリッド設定がすべて復元
⚠ 注意: JPEG を別アプリで再保存すると埋め込みデータが失われる場合があります。 オリジナルの撮って出しを保管してください。

🌅 環境 (疑似HDRI)

cbar の「🌅 環境」ボタンから。空の色とシーン全体の色味を一括変更。

  • 9 プリセット: ☀ 昼 / 🌅 朝 / 🌇 夕方 / 🌃 夜 / ☁ 曇り / ⛅ 薄曇 / 🌧 雨 / 🌌 黄昏 / なし
  • 回転スライダ 空を 0–360° 回せる (光の方向を合わせる)
  • 強度スライダ 0–200% (100% で標準, 200% で強めにライティング感)

選んだ環境で splat やモデルにも乗算ティントがかかり、夜なら全体が暗青、夕方なら暖色寄りに見えます。

🗺 ミニマップ

cbar の「🗺 マップ」をクリックで表示/非表示。

  • 右クリック長押し → 離した位置に瞬間移動
  • 表示距離: 10 / 50 / 100 / 500 m (デフォルト 100m)
  • クリックでマップ拡大表示

🖥 画質と速度

cbar の「🖥 画質」で描画品質を切替。

  • 低品質 = 動作軽快 (大きい PLY や非力 GPU 用)
  • 標準 / 高品質 = 通常使用
  • カメラ動作中は自動的に解像度を 65% に下げ、停止後に復元 (Spark のソート遅延対策)

💾 プロジェクト保存

  • 左上の 📁 保存 ボタン → ZIP でダウンロード (シーン全体 + カメラ設定)
  • Ctrl+S でも同じ
  • 左上 📥 読込 または起動画面「📦 プロジェクトを開く」で復元
  • 「▶ 編集を続ける」(起動画面) は前回の編集状態を継続

❓ よくある質問

Q. 読み込んだシーンが上下逆さま

A. レイヤー選択 → 表示調整の「X軸反転 / Y軸反転 / Z軸反転」を試してください。

Q. カメラを振ったとき画面端が黒い / にじむ

A. Spark のソート遅延です。動作中は自動で解像度が下がり、停止すれば復元されます。 どうしても気になる場合は cbar の「🖥 画質」を一段下げてみてください。

Q. 測定が表面を貫通して奥に置かれる

A. キャッシュ密度を上げ済 (最大 2M 点)。それでもズレる場合は、選択中のレイヤーが 正しく メイン splat を含んでいるか確認してください。

Q. 撮った JPEG をサルベージしたら復元されない

A. 別ソフトで再エンコードされた可能性があります。ロケハン3Dの撮って出しのみ対応します。

Q. F値や ISO がない

A. 仮想カメラなので被写界深度や露出はありません。色温度 (WB) のみ画作りに反映します。

Q. もう一度マニュアルを開きたい

A. 画面下中央の「❓」ボタンからいつでも開けます。

🚀 5-Minute Quickstart

Locahun3D is a viewer for loading 3D scans (3DGS) for location scouting, storyboards and previz. Start with these three steps.

  1. Load — drag a 3DGS file (.splat / .ply, etc.) onto the “📂” area on the start screen, or hit “Demo scene” to try it.
  2. Move — right-drag the mouse to look around; WASD to move.
  3. Use — bottom buttons: Camera / Sun / Measure. Top button: Add object. Manage everything in the left layer panel.

🎮 Moving the camera

  • Look around: right-drag the mouse
  • Move: Wfwd / Sback / Aleft / Dright / Qdown / Eup
  • Speed: mouse wheel (or the slider in the quality panel)
  • Reset view: “🏠 Camera reset” at the top

On phone / iPad, use the bottom-left joystick to move and drag the screen to look around.

📁 Layers & folders

Loaded files and added objects appear in “Scene layers” on the left.

  • 👁 toggle visibility
  • ✏️ rename (double-click also works)
  • 🗑 delete (Ctrl+Z to undo)
  • drag to reorder / drop into a folder
  • 🔒 Lock a folder to disable moving, editing and deleting its contents (prevents accidents). Click again to unlock.

⇆ Move / rotate / scale

Select an object to get the 3D handles (gizmo) and numeric inputs.

  • arrows = move / rings = rotate / cubes = scale
  • switch coordinate space with “Local / World” (new & imported objects default to Local)
  • position / rotation / scale / size inputs step by 0.01 for fine adjustment
  • on iPad / phone you can drag the gizmo (arrows, rings, pivot) with your finger

＋ Adding objects

Pick a type from “📦 Add object” at the top, then long-press (left-click) in the scene to place it (hold to aim, release to confirm; Esc to cancel).

  • 📦 Cube — size reference / blocking
  • 🟠 Event — a point of interest; you can attach an image
  • 👤 Figure — a human; adjust height to check scale & eye level, and pose it by selecting joints to rotate or pulling the IK handles
  • 🛣 Path — a 4-point area with a centre label (e.g. parking bays). Labels can be multi-line.

📐 Distance measure

Turn on “📐 Measure” at the bottom. Measures the real distance between two points on the scan surface.

  • long-press to preview a point, release to confirm (point A → point B)
  • “+ Measure height” adds a 3rd point for height
  • fine-tune markers with the arrows (axis) or the XZ plane; 🗑 to clear
  • on iPad / phone you can also drag the arrows and markers with your finger
  • distance, height & components read out in 0.01 m (1 cm) steps

📷 Camera tool / animation

Camera tool (“📷 Camera” at the bottom): set a real lens, sensor and aspect ratio, show the framing box and grid, and capture JPEGs with camera metadata burned in.

Camera animation (“🎞 Animation” at the top): add several camera keys to build a path and export an MP4. Motion can be Ease in/out or Linear.

☀ Sun & weather

“☀ Sun” at the bottom: reproduces the sun position and lighting mood from date, time and prefecture.

  • scrub the time slider from morning to night
  • change the sky mood with weather (clear / cloudy / rain)
  • “🌐 Get live forecast” follows real online weather (optional)

🚶 Avatar walk / 📱 AR

Avatar walk (“🚶 Avatar walk” at the top): explore the scene from a person’s eye level. Move the avatar with WASD; look around as usual. Great for checking sightlines, eye level and sense of scale. Click again to exit.

AR mode (“📱 AR” at the top; phone / tablet only): overlays the 3DGS on the rear-camera feed so you can tilt the device to look around — handy for checking the real-scale overlay on location.

Note: on iOS, first enable “Settings > Safari > Motion & Orientation Access”, then reload the page to use AR.

💾 Save & load

  • Zip save (top-left): saves the whole scene into a single ZIP — layers, objects, camera and settings all restore.
  • Load (“📁 Load” above the layer panel): add a ZIP or a 3DGS / model.
  • 🎥 Save camera: stores the current camera position as a layer.

❓ FAQ & shortcuts

  • Scene upside-down / sideways → select the layer and use “Flip axis” or the angle inputs.
  • Running slow → set the quality chip (top-right) to “Low”. Rendering is capped at 60 fps.
  • Language → toggle “EN / 日本語” at the top-right.

Shortcuts: Ctrl+Z undo / Ctrl+Y redo / Ctrl+D duplicate
